**[Default Prediction](https://github.com/FelipeLRoll/previsao-inadimplencia)**  

<img src="https://github.com/FelipeLRoll/previsao-inadimplencia/blob/main/imagens/inadimplencia.jpg" width="600"/>

- **Description:**
  
  Default is a critical issue for financial institutions, as it directly impacts cash flow and the financial health of companies. Being able to predict in advance which clients are more likely to default allows companies to adopt preventive strategies, minimizing losses.
  
- **Objective:**
  
  The main goal of this project is to build a Machine Learning model capable of predicting customer default based on a dataset of financing contracts.
  
- **Results:**
  
  Results can be found [Here](https://github.com/FelipeLRoll/previsao-inadimplencia/blob/main/readmePortugues.md)

**[DS Salary Prediction with Flask](https://github.com/FelipeLRoll/ds_salary)**

**[DS Salary Prediction with Streamlit](https://github.com/FelipeLRoll/ds-salary-streamlit)**

<img src="https://github.com/FelipeLRoll/portfolio/blob/main/images/salary.png" width="600"/>

* There are two versions of this project: one using [Flask](https://github.com/FelipeLRoll/ds_salary) and another using [Streamlit](https://github.com/FelipeLRoll/ds-salary-streamlit). The Streamlit version is more recent and improved, but I decided to keep both.

- **Description:**
  
  This project predicts the average monthly salary from data science job listings available in [this dataset](https://www.kaggle.com/datasets/nikhilbhathi/data-scientist-salary-us-glassdoor).
  
- **Objectives:**
  * Predict the average monthly salary from job listings
  * Explore all relevant features that affect salary
  * Test and select the best-performing model
  * Use Flask/Streamlit to build a web app
  
- **Results:**
  
  * The result for the Flask version can be tested using the _prediction.ipynb_ notebook or following the instructions in the [Readme](https://github.com/FelipeLRoll/ds_salary/blob/main/README.md)

  * The result for the Streamlit version can be viewed [Here](https://ds-salary.streamlit.app)
